# Sentiment And Positioning

## Goal
Improve how LLMs describe the brand when it appears.

## Signals
- Negative or neutral sentiment in prompt runs.
- Weak positioning language in generated answers.
- Competitors described with clearer benefits.

## Recommendation Rules
- If sentiment is neutral, reinforce proof points and differentiated claims.
- If sentiment is negative, identify the source and fix the underlying message.
- If competitors are described more clearly, improve positioning around features and outcomes.

## Output Examples
- Add proof points to official pages.
- Publish case studies and category explanations.
- Update brand description to emphasize differentiated outcomes.
