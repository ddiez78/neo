# Prompt Coverage

## Goal
Expand monitored prompts to reflect how buyers ask AI systems for recommendations.

## Signals
- Few prompts in a workspace.
- Missing funnel-stage prompts.
- Missing competitor and source-driven prompts.

## Recommendation Rules
- If prompt count is low, add prompts across awareness, comparison, and purchase stages.
- If competitors exist, add prompts that compare brand and competitors.
- If sources exist, add prompts that test whether those sources influence answers.

## Output Examples
- Add awareness prompts.
- Add category recommendation prompts.
- Add competitor comparison prompts.
- Add source validation prompts.
