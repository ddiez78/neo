# Content Gap Analysis

## Goal
Find content gaps that reduce AI answer visibility.

## Signals
- Prompts without brand mentions.
- Competitor co-mentions.
- Missing content for comparison, pricing, alternatives, use cases, and industry terms.

## Recommendation Rules
- If prompts exist but brand mentions are weak, create content aligned with those prompts.
- If competitors appear frequently, create comparison and alternative content.
- If prompt coverage is narrow, add prompts across funnel stages.

## Output Examples
- Create "best tools for..." pages.
- Create "brand vs competitor" comparison pages.
- Create glossary and educational pages for category terms.
