# Bing Search APIs retiring on August 11, 2025

Feedback

 Summarize this article for me 

## In this article

Bing Search APIs will be retired on August 11, 2025. Any existing instances of Bing Search APIs will be decommissioned completely, and the product will no longer be available to be used or new customer signup.

Customers are encouraged to migrate to [Grounding with Bing Search as part of Azure AI Agents](https://learn.microsoft.com/en-us/azure/ai-services/agents/how-to/tools/bing-grounding?view=azure-python-preview&tabs=python&pivots=overview). Grounding with Bing Search allows Azure AI Agents to incorporate real-time public web data when generating responses with an LLM.

[](https://learn.microsoft.com/en-us/lifecycle/announcements/bing-search-api-retirement#additional-resources)
